# EdgeAI SDK Test Scripts

## Scripts

- **`run_sdk_tests_and_report.sh`** - Run all tests and generate report
- **`run_sdk_tests.sh`** - Run tests only (no report)
- **`generate_sdk_test_report.sh`** - Generate report from logcat

## Usage

From EdgeAI root:
```bash
./run_tests.sh
```

See [docs/maintainers/testing.md](../docs/maintainers/testing.md) for details.
